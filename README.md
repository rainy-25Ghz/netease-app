# 网易云音乐 react ts版本


## 1.安装netease api
```shell
$ git clone https://github.com/Binaryify/NeteaseCloudMusicApi.git

$ npm install / yarn 
```

## 2. 启动后端服务

```shell
$ PORT=4000 node app.js
```
## 3. clone至本地，运行网易云音乐
``` shell
$ git clone https://github.com/rainy-25Ghz/netease-app
$ npm install / yarn
```
``` shell
$ npm/yarn start 
```
